# angel_api


A handy NodeJS API for every endpoint needed available on the internet.

## Tools

### [LoopBack.io](https://loopback.io/doc/en/lb4/todo-list-tutorial-repository.html)

Commands:

```bash
$ lb4 angel_api
$ lb4 model
$ lb4 datasource
$ lb4 repository
```

### More
